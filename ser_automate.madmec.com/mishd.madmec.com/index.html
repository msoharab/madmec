Help Desk
 

Welcome to the help Desk..

1. User

User has two sub features:

1.1 Add User
1.2 List User
 

1.1 Add User

here you can add users based upon their type, follow are the basic details you have to fill

# Usertype
# Username
# Product
# Email
# Mobile Number
# Bank Account
# Land Line Number
# Address
 
 

1.2 List User

here you can view all users Details as well as having option to edit/add details,

2. Stock

Stock has Three sub features:

2.1 Add Item
2.2 Add Stock
2.3 Available Stock
 
 

2.1 Add Item

here you can add new Item in WareHouse for that have to fill below details

# Item Name
# Minimum Criteria
 
 

2.2 Add Stock

for Adding you have to select Item Name and enter quantity value.

 
 

2.3 View Stock

here you can see Avaliable Stock of all type of items in warehouse

3. Material Order

Material Order has Four sub features:

3.1 Create Order
3.2 Add Item
3.3 View Order
3.4 Item Supplied
 
 

3.1 Create Order

you can Create Order for that have to select/ fill below fields.

# Vendor/Supplier
# Item
# Date of Order
# Expected Date of Deliver
# Quantity
 
 

3.2 Add Item

here you can add more item to Existing Order what you Create in 3.1 module as well as having option to Edit/Delete Add Items, as soon as you click PDF Button it will generate Material Order Report in PDF file Format

 
 

3.3 View Order

here you can view Generated Material Order Items and having option to Update stock-in, once you received all the Items the Order it won't appear

 
 

3.4 Item Supplied

You Can view Item Supplied Details of all Created Orders.

5. Accounts

Accounts has Five sub features:

5.1 Incoming
5.2 Outgoing
5.3 Petty Cash
5.4 Due
5.5 Fallow ups
 
 

5.1 Incoming

Incoming has three sub features:

5.1.1 Add Collection
5.1.2 Project Collection
5.1.3 List Collection
 
 

5.1.1 Add Collection

You Can add Collection for that you have to select/fill below fields.

# Select Payer
# Date of Collection
# Mode of Payment
# Amount
# Remark
 
 

5.1.2 Project Collection

You Can add Project Collection for that you have to select/fill below fields.

# Select Client
# Select Project
# Date of Collection
# Amount
# Remark
# Due Date
# Fallow Date's
 
 

5.1.3 List Collection

Displaying Collection Transaction History according to date wise

 
 

5.2 Outgoing

Outgoing has two sub features:

5.2.1 Add Payment
5.2.2 List Payments
 
 

5.2.1 Add Payment

You Can issue Payments for that you have to select/fill below fields.

# Select Payee
# Date of Payment
# Mode of Payment
# Amount
# Remark
 
 

5.2.2 List Payments

Displaying Payment Transaction History according to date wise

 
 

5.3 Petty Cash

Petty Cash has two sub features:

5.3.1 Add Cash
5.3.2 Petty Cash History
 
 

5.3.1 Add Cash
